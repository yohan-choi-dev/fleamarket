# FleaMarket - Backend repository

### Quick start

1. Installing dependencies: `npm install`
2. Running the server
    - Development: `npm run start`
    - Production: `npm run production`

### API Endpoints (`/api`)

- `/auth/signup`
- `/auth/login`
- `/items`
  - `/items?name=Logitech`
  - `/items?user=51`
